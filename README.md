# Интерполяция функции: сравнение методов

Этот проект реализует сравнение различных методов интерполяции функции и визуализацию погрешностей.

## Описание

Программа сравнивает три метода интерполяции:
1. **Полиномиальная интерполяция с равномерными узлами**
2. **Полиномиальная интерполяция с узлами Чебышева**
3. **Кубическая сплайн-интерполяция**

Для каждого метода вычисляется максимальная погрешность интерполяции и строится график зависимости погрешности от количества узлов интерполяции.

## Функциональность

- Интерполяция функции `f(x) = x^cos(x)` на отрезке [2, 10]
- Построение графиков погрешностей для каждого метода
- Анимация изменения погрешностей при увеличении количества узлов
- Сравнение методов в логарифмическом масштабе

## Использование

### Настройка параметров

Перед запуском можно изменить основные параметры в коде:

```python
def f(x):
    return x**np.cos(x)  # Замените на вашу функцию

a = 2     # Левая граница интервала
b = 10    # Правая граница интервала
k = 1000  # Количество точек для оценки погрешности
m = 7     # Максимальное количество узлов интерполяции
```

### Запуск программы

1. Запустите скрипт
2. Программа последовательно покажет графики погрешностей для разного количества узлов (от 2 до m)
3. Нажмите Enter для перехода к следующему количеству узлов
4. После просмотра всех шагов отобразятся итоговые графики:
   - Сравнение исходной функции и интерполирующих кривых
   - Зависимость логарифма погрешности от количества узлов
